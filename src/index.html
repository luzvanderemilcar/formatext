Copy text from JW Library

Copy Bible verse text from JW Library "Parallel translations" panel can result in a poor readable text. So, before passing the text inside any TTS program, this tool helps format it in a better way.

How to use

  1. Type into the textbox what you want to reformat or paste it using the Paste button
  2. Preview it in the section below
  3. Tap the Clear button to empty the textbox
  4. Double tap the preview text to copy it into your clipboard, or tap the Copy button at the very bottom of the preview

Note

Check whether your browser has clipboard read and write permissions, if any problem copying or pasting content
